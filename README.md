# Stardew_Valley_ESP
This is an agent that helps you 100% the game as efficiently as you can by informing the user day to day actions and opportunities when certain conditions are met. This agent does not aim to highlight entities, and does not give you wallhacks. The agent will doing the following but not exclusive to: Provide you step by step instruction to complete tasks, keep track of current tasks, alert you of tasks close to completion, and many more related features.

When it makes sense the agent will execute scripts, powershell, or what need be to gather the data it needs to perform its task.
